What to do when things are not normal?

  • Step1. Feel uncomfortable

  • Step2. Consider data transformations or Generalised Linear Models

    • Transforming may be better when you have problems with variance, as well as normality.
    • GLM's are likely to be better when you have count data, proportion data, or presence/absence data.
    • GLM results are often easier to interpret (I think)

  • Step3. Realise that having non-normal data is pretty normal in ecology.

Generalised Linear Models

  • Model response data with specified non-Gaussian (non-normal) response
  • Can accommodate continuous and categorical explanatory variables
  • Can account for variable interactions and quadratic terms
  • Response should be linear
  • Outliers are still a problem, and variance can still cause issues

In R we must specify a distribution type and a link function.

Some common distributions for continuous data

Normal distribution

  • unbounded
hist(rnorm(1000), col = "blue")

Gamma distribution

  • Bound at zero
  • positive numbers only
hist(rgamma(1000, shape = 1), col = "red")

Poisson distribution

  • Bound at zero
  • Positive numbers and zeros
  • Integers only
hist(rpois(1000, lambda = 4), col = "green")

Negative binomial distribution

  • Bound at zero
  • Positive numbers and zeros accepted
  • More dispersed than poisson
hist(rnbinom(1000, mu = 4, size = 10), col = "grey")

Binomial distribution

  • For data with only two categories, or proportional/percentage data
    • presence/absence
    • survival data
hist(rbinom(1000, size = 10, prob = 0.5), col = "purple")

  • Link function: logit or clogit
